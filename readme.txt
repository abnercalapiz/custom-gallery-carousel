== Description ==

Custom Gallery Carousel is a responsive image gallery built with Owl Carousel. It lets users define custom gallery fields and integrate seamlessly with WooCommerce galleries, offering optimized settings for mobile, tablet, and desktop devices.

== Features ==

* Gallery carousel with thumbnail navigation
* Adjustable item count for different screen sizes (desktop, tablet, mobile)
* Lightbox functionality for full-screen image viewing
* Keyboard navigation support for lightbox (arrow keys and Escape)

== Installation ==

1. In your admin panel, go to Plugins -> click the 'Add New' button.
2. Click 'Upload Plugin', choose file and install.
3. Click on the 'Activate' button to use Custom Gallery Carousel right away.
